Your Guide to Using Our API

GET /api/v1/users

Retrieve a list of all users.

curl -X GET https://api.example.com/api/v1/users

Response:


[
  {
    "id": 1,
    "name": "John Doe",
    "email": "john.doe@example.com"
  },
  {
    "id": 2,
    "name": "Jane Smith",
    "email": "jane.smith@example.com"
  }
]

POST /api/v1/users

Create a new user.

curl -X POST https://api.example.com/api/v1/users -d '{
  "name": "New User",
  "email": "new.user@example.com"
}'

Response:


{
  "id": 3,
  "name": "New User",
  "email": "new.user@example.com",
  "message": "User created successfully."
}

PUT /api/v1/users/:id

Update an existing user.

curl -X PUT https://api.example.com/api/v1/users/1 -d '{
  "name": "Updated User",
  "email": "updated.user@example.com"
}'

Response:


{
  "id": 1,
  "name": "Updated User",
  "email": "updated.user@example.com",
  "message": "User updated successfully."
}

DELETE /api/v1/users/:id

Delete a user by ID.

curl -X DELETE https://api.example.com/api/v1/users/1

Response:


{
  "message": "User deleted successfully."
}
